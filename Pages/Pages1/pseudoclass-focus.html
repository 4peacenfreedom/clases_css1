Pseudo clase :focus

La pseudo-clase :focus en CSS se utiliza para aplicar estilos a un elemento cuando recibe el foco de atención del usuario. Un elemento recibe el foco cuando el usuario lo selecciona para interactuar con él, generalmente mediante el uso del teclado o el ratón. Por ejemplo, cuando haces clic en un campo de entrada de texto, este campo se enfoca y puedes comenzar a escribir en él.

La pseudo-clase :focus es muy útil para proporcionar feedback visual al usuario cuando interactúa con formularios o elementos interactivos de una página web. Esto puede ayudar a mejorar la usabilidad y la accesibilidad del sitio.

Aqui hay un ejemplo:


            /* Ejemplo de codigo CSS para realizar el :focus  */
            
            input:focus {
              border: 2px solid red; /* Cambia el borde del campo de entrada cuando está enfocado */
            }

Y como se funcionaria? haz click en el area de texto y veras como el borde se vuelve rojo
