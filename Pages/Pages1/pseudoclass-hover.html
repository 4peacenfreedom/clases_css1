Pseudo clase :hover

La pseudo-clase :hover en CSS se utiliza para aplicar estilos a un elemento cuando el usuario coloca el cursor sobre él. Es una forma de agregar interactividad y feedback visual a los elementos de una página web.

Cuando el cursor está sobre un elemento y se activa la pseudo-clase :hover, se pueden aplicar estilos adicionales al elemento, como cambios en el color, tamaño, posición o cualquier otra propiedad CSS.

Aqui hay un ejemplo:


            /* Ejemplo de codigo CSS para realizar el :hover  */

            a {
                color: blue; /* color predeterminado */
              }
              
              a:hover {
                color: red; /* color cuando el cursor está sobre el enlace */
              }
