Pseudo clase :active

La pseudo-clase :active en CSS se utiliza para aplicar estilos a un elemento mientras está siendo activado por el usuario. En general, se activa cuando un usuario hace clic en un elemento, aunque también puede ser activado por otros eventos dependiendo del tipo de elemento.

Por ejemplo, cuando haces clic en un botón, enlace o cualquier otro elemento interactivo, el navegador aplica temporalmente los estilos definidos para :active a ese elemento.

Aqui hay un ejemplo:


            /* Ejemplo de codigo CSS para realizar el :active  */
            
            a:active {
              color: green; /* Cambia el color del enlace cuando está siendo activado */
            }

En este ejemplo, cuando un usuario hace clic en un enlace , el color del texto del enlace cambiará a verde mientras está siendo activado. Una vez que el usuario suelta el clic, el color volverá a su estado original.
